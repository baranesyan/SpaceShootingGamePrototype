# 🚀 3D Space Shooter Game Prototype

A small fast-paced top-down space shooter inspired by classics, where the player fights off waves of enemies and survives as long as possible.

## 🎮 Gameplay

The player controls a spaceship and fights against incoming enemy waves.
The goal is to survive, destroy enemies, and achieve the highest score possible.

## 🧠 Core Mechanics

* Smooth top-down spaceship movement
* Shooting system with projectiles
* Multiple enemy types with different speed 
* Restart/reset system for experimentation
* Collision and damage system
* Score tracking system
* Endless enemy spawning
---
